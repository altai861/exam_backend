Functional Requirements: 
1. Хэрэглэгч бүртгүүлнэ, нэвтэрнэ
2. Шалгалтуудыг болон шалгалтын дэлгэрэнгүйг харж болно.
3. Шалгалт шинээр үүсгэж болно.
4. Шалгалт засаж болон устгаж болно.
5. Шалгалт өгж болно.
6. Шалгалт тус бүрийн хэрэглэгчдийн авсан оноог жагсаалтаар харж болно. Бас өөрийн өгсөн бүх шалгалтын оноог харж болно.
